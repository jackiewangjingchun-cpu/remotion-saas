# VideoGen Pro - AI 自动化视频生成平台

## 核心功能

### 1. 智能内容抓取
- **网页抓取**：输入 URL 自动提取标题、正文、图片
- **RSS 订阅**：自动抓取新闻源生成日更视频
- **API 接入**：连接社交媒体、电商平台
- **文档解析**：PDF/Word/Excel 转视频

### 2. AI 内容处理
- **文本总结**：长文 → 30秒脚本
- **情感分析**：自动匹配视频风格
- **关键词提取**：生成标签和描述
- **多语言翻译**：一键生成多语言版本

### 3. 自动化模板
| 模板 | 输入 | 输出 |
|------|------|------|
| 新闻快讯 | 新闻 URL | 30秒新闻视频 |
| 数据日报 | 数据表格 | 动态图表视频 |
| 产品推广 | 商品链接 | 营销视频 |
| 生日祝福 | 人物信息 | 个性化视频 |
| 课程总结 | 文档/PPT | 知识短视频 |

## 技术栈升级

```
前端：Next.js + Tailwind
├── 用户选择模板
├── 输入信息源
├── 预览编辑
└── 下载管理

后端：Node.js / Python
├── 内容抓取服务 (Playwright / Cheerio)
├── AI 处理 (Kimi API / OpenAI)
├── 视频渲染 (Remotion Lambda)
├── 任务队列 (Bull / Redis)
└── 文件存储 (S3 / R2)

数据库：PostgreSQL
├── 用户表
├── 模板表
├── 任务队列
├── 视频记录
└── 订阅配置
```

## API 设计

### 创建自动化任务
```http
POST /api/jobs
{
  "templateId": "news-daily",
  "source": {
    "type": "rss",
    "url": "https://techcrunch.com/feed/"
  },
  "schedule": "0 9 * * *",  // 每天9点
  "output": {
    "format": "mp4",
    "resolution": "1080x1920"
  }
}
```

### 单次生成
```http
POST /api/generate
{
  "templateId": "product-showcase",
  "source": {
    "type": "url",
    "url": "https://amazon.com/product/xxx"
  },
  "aiProcessing": true
}
```
